# Phase 1: Cost Optimization Simulation (C Language)

## 目的
本シミュレーションの目的は、理論の中核である「異種エージェント（人間、LLM、ロボット）間のコストと能力差を利用した全体最適化」が、数理的に成立することを小規模モデルで検証することである。

## 理論的背景
社会ハミルトニアン $H$ の最小化プロセスにおいて、各エージェントは以下の特性を持つと仮定する。

1. 人間 (Human):
    * 創造性: 高
    * コスト: 高 (Well-being維持のため)
    * エラー率: 中
    * 特性: $W_{human}$ (幸福度) がハミルトニアンの減少項として機能する。
2. LLM (Large Language Model):
    * 創造性: 中 (確率的)
    * コスト: 極低 (トークン単価)
    * エラー率: 変動 (幻覚)
3. ロボット (Robot/Legacy Automation):
    * 創造性: 低 (定型処理のみ)
    * コスト: 中 (初期投資・メンテ)
    * エラー率: 低

## 実装計画
- 言語: C (Standard C99/C11)
- アルゴリズム:
    - ランダムなタスクセット（論理的タスク、創造的タスク、肉体作業）を生成。
    - 各エージェントにタスクを割り当てた際の総コスト $H$ を計算。
    - 最適な割り当てを探索する（全探索 or 貪欲法・焼きなまし法）。
- 検証項目:
    - 「LLMのコストが低下すると、人間の役割がより創造的タスクにシフトする（パレート最適）」ことが再現できるか。
    - 「人間が何もしない（UBI受給）」ほうが全体最適になる条件が存在するか。
